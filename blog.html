Local Storage -লোকালস্টোরেজ সম্পত্তি আপনাকে একটি স্থানীয় স্টোরেজ অবজেক্টে অ্যাক্সেস করার অনুমতি দেয়। লোকালস্টোরেশন সেশনস্টোরেশনের সমান। পার্থক্যটি হ'ল লোকালস্টোরেজে থাকা ডেটাটির কোনও মেয়াদোত্তীর্ণের সময় নেই এবং সেশনসটোরেজে সংরক্ষিত ডেটা ব্রাউজার সেশন শেষে (অর্থাৎ ব্রাউজারটি বন্ধ হয়ে গেলে) সাফ হয়ে যায়।
Session Storage -সেশন স্টোরেজ সম্পত্তি আপনাকে একটি সেশন স্টোরেজ অবজেক্টে অ্যাক্সেস করার অনুমতি দেয়। লোকালস্টোরেশনে সঞ্চিত ডেটাটির কোনও মেয়াদোত্তীর্ণ সময় সেটিং নেই, এবং সেশনসটোরেজে সংরক্ষিত ডেটা পৃষ্ঠা সেশনের শেষে সাফ হয়ে যায় except ব্রাউজারটি খোলা থাকাকালীন পৃষ্ঠা সেশনটি বজায় রাখা হয় এবং পৃষ্ঠাটি পুনরায় লোড বা পুনরুদ্ধার করা এখনও মূল পৃষ্ঠা সেশনটি বজায় রাখে।

Global Scope - যদি কোনো variable -কে কোনো একটি function অথবা কোনো { } - এর মধ্যে ডিক্লেয়ার না করে কোনো 'open space' -অর্থাৎ Globaly ডিক্লেয়ার করা হয়, তাহলে সেটিকে Global Scope বলা হবে। এক্ষেত্রে var, const অথবা let -যে keyword ই বাবহার করে ডিফাইন করা হোক না কেন, সেই variable কে কোডের যেকোনো জায়গা থেকেই access করা যাবে।.
Block Scope- ES6' হচ্ছে JavaScript এর Modern Version)। Block Scope এর মূল বৈশিষ্ট্য হচ্ছে { } -block এর মধ্যে যদি আমরা কোনো variable ডিক্লেয়ার করি তাহলে সেই variable কে block এর বাইরে থেকে access করা যায় না।

ইভেন্ট লুপের একটি সহজ কাজ রয়েছে - কল স্ট্যাক এবং কলব্যাক কিউ নিরীক্ষণ করা। কল স্ট্যাক খালি থাকলে, ইভেন্ট লুপ সারি থেকে প্রথম ইভেন্টটি নিয়ে যাবে এবং এটিকে কল স্ট্যাকের দিকে ঠেলে দেবে, যা কার্যকরভাবে এটি চালায়। এই ধরনের পুনরাবৃত্তিকে ইভেন্ট লুপে টিক বলা হয়।
